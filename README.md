##Usage
**Install dependencies with virtual env**

    $ mkdir project
    $ cd project

create a file name `requirements.txt` with content

    oauth

Create virtual environment and instal requirements

    $ virtualenv venv
    $ . venv/bin/activate
    $ pip install -r requirements.txt
